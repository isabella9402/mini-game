# 🐱 나비의 옥상 (Nabi's Rooftop)

> 한 손으로 즐기는 **지붕 위 그네타기** — 고양이 **나비**가 실을 걸고(길게 누르기) 놓으며(떼기) 서울 지붕을 오른쪽으로 날아갑니다. **Stickman Hook** 스타일의 진자(swing) 물리를 그대로 담은 **가로 16:9** 미니게임입니다.

`index.html` 하나만 열면 바로 실행됩니다. 빌드 · 설치 · 외부 이미지/사운드 파일이 전혀 필요 없습니다.

---

## ▶️ 플레이 방법

| 플랫폼 | 조작 |
|---|---|
| 📱 모바일 (가로) | 화면을 **길게 누르면** 가장 가까운 앞쪽 실(●)에 걸립니다 → **떼면** 접선 방향으로 날아갑니다 |
| 💻 PC | **마우스 왼쪽 버튼** 또는 **Space**를 누르면 걸기 / 떼면 날아가기 |

- 조작은 **딱 하나** — 누르기/떼기 타이밍만으로 진자 운동의 힘을 모아 멀리 날아갑니다.
- 앞쪽(오른쪽·위 방향)의 실이 **빛나는 원**으로 자동 표시됩니다. 누르면 그 실에 걸립니다.
- **아래 골목길(도로)에 닿으면 게임 오버.** 단, 옥상 사이에 널린 **이불(트램펄린)** 에 떨어지면 위-앞쪽으로 튕겨 살아남습니다.

### 아이템
- 🐟 **붕어빵** — 날아가며 먹으면 +점수
- ⚡ **츄르** — 먹으면 **피버 타임!** (약 5초, 중력 감소 · 점수 ×2 · 화려한 배경)

거리(m)가 곧 점수이며, 게임 오버 화면에서 **결과 복사** 로 카톡에 자랑할 수 있습니다.

## 🚀 실행 & 배포

### 로컬에서 바로 실행
`index.html` 을 브라우저로 열기만 하면 됩니다. (모바일은 **가로**로 돌려주세요 — 세로면 `가로로 돌려주세요` 안내가 뜹니다.)

### GitHub Pages 배포
1. 이 저장소의 **Settings → Pages** 로 이동합니다.
2. **Build and deployment → Source** 를 `Deploy from a branch` 로,
   **Branch** 를 `main` / `/ (root)` 로 설정하고 저장합니다.
3. 잠시 후 게임이 아래 주소에서 열립니다:

   ```
   https://isabella9402.github.io/mini-game/나비의%20옥상/
   ```

   > 폴더 이름에 한글·공백이 있으므로 URL에서는 공백이 `%20` 으로 인코딩됩니다. 더 짧은 주소를 원하면 폴더를 `nabi-rooftop` 같은 영문 이름으로 옮겨도 됩니다.

`결과 복사` 기능의 공유 링크는 현재 페이지 주소(`location.href`)를 그대로 사용하므로, 배포한 뒤엔 자동으로 배포 주소가 붙습니다.

## 🎮 기능 요약

- **가로 16:9 고정** — 논리 해상도 640×360, `devicePixelRatio` 대응(High-DPI), 화면비가 다르면 레터박스.
- **프레임 독립 물리** — 고정 타임스텝(`STEP = 1/120s`)으로 60Hz·120Hz·프레임 드랍에서도 동일한 진자 운동.
- **Verlet 진자 물리** — 거리 제약(distance constraint)으로 걸 때는 낙하 운동량이 자연스럽게 회전으로, 뗄 때는 접선 방향으로 그대로 보존됩니다. (`theta''` 방정식 없이 운동량 보존이 저절로 성립)
- **오른쪽 자동 조준** — 진행 방향(오른쪽·위) 원뿔 안의 가장 좋은 실을 골라 발광 링으로 안내.
- **끝없는 절차적 맵** — 항상 "걸어서 갈 수 있는" 간격으로 실/옥상 생성, 거리에 따라 난이도 상승, 초반은 쉽게(트램펄린 온보딩).
- **전진 전용 카메라** — 오른쪽으로만 전진, 높이 솟으면 위로 팬. 아래 도로 아래로는 따라가지 않아 추락이 명확.
- **HUD / 피버 / TOP 3 / 결과 복사** — `localStorage`(`nabi_roof_best`)에 최고 기록 3개 저장, Web Audio 효과음(파일 불필요, 🔊/🔇 토글).

## 🔧 튜닝 (코드 상단 `CONFIG`)

`index.html` 최상단 `CONFIG` 객체만 고치면 손맛을 조절할 수 있습니다.

| 값 | 의미 |
|---|---|
| `GRAVITY` | 중력 (px/s²) — 클수록 빨리 떨어짐 |
| `ROPE_MIN` / `ROPE_MAX` | 실 길이 범위 |
| `SCAN_R` / `CONE_DEG` | 자동 조준 반경 / 원뿔 반각 |
| `BOUNCE_IMPULSE` | 트램펄린 튕김 세기 |
| `CAM_ANCHOR_X` | 화면에서 고양이 가로 위치(0~1) |
| `STREET_Y` | 도로(죽는 선)의 월드 Y |
| `FEVER_*` | 피버 지속시간 · 중력 · 점수 배율 |
| `HOOK_MIN_GAP` | 실 사이 최소 간격 |
