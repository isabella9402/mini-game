# 길냥이 붕어빵 런 (Street Cat's Bungeoppang Run)

한 손으로 즐기는 무한 길 건너기 — 서울 길고양이 **나비(Nabi)** 가 붕어빵을 주우며
도로 · 지하철 · 한강을 건너는 크로시로드 스타일 캐주얼 게임.

바닐라 **HTML5 Canvas 2D**로만 제작 — 빌드 도구 · 외부 이미지/오디오 파일이 **전혀 없습니다.**
`index.html`을 더블클릭하면 바로 실행됩니다. (효과음은 Web Audio로 생성, 그래픽은 Canvas로 렌더링)

## 파일 구조
```
street-cat-run/
├── index.html   # DOM, Galmuri 폰트(jsDelivr), 캔버스
├── style.css    # 9:16 레터박스 · 캔버스 중앙 정렬
└── game.js      # 게임 로직 전부 (루프/입력/맵생성/카메라/충돌/오디오/저장)
```

## 규칙
- **점수** = 가장 멀리 나아간 칸 + 아이템 보너스. 뒤에서 카메라가 계속 밀어오니 멈추면 안 됩니다.
- 🐟 **붕어빵** +점수 · 🪙 **500원 코인** 수집 · 🐟 **한강 물고기** +보너스
- 🐱 **츄르** → **피버 타임!** 5초간 무적 + 점수 x2
- **도로/지하철**: 차·지하철에 부딪히면 게임 오버 (지하철은 빨간 경고등이 깜빡인 뒤 지나갑니다)
- **한강**: 통나무 위에서만 살 수 있고, 통나무를 타고 화면 밖으로 떠내려가면 게임 오버
- 최고 점수 **TOP 3**는 로컬에 저장됩니다.

## 재사용한 코드 / 에셋 (Reuse)
이 저장소의 이전 Three.js 게임(*꼬마 도깨비의 모험*)에서 아래를 가져와 재사용했습니다:
- **Web Audio `SoundFX`** 클래스 — 파일 없는 효과음 생성기
- **localStorage 안전 래퍼 `LS`** — 샌드박스에서도 crash 없는 최고점수 저장
- tap/swipe 입력, `camera = max(auto, follow)`, 강물 뗏목 타기, 난이도 램프,
  피버 모드 등 **게임플레이 로직 패턴** 및 파스텔 한국풍 UI 감성
- 테마 연속성: 이전 게임의 **고양이 주인공**과 **붕어빵(3D_붕어빵)** 소재

> v1은 사양(§1)에 따라 외부 이미지 파일을 쓰지 않습니다. 3D `.jpg` 에셋은 3D 렌더 배경이 있어
> 2D 캔버스에는 부적합하고 "링크 열면 바로 실행" 원칙과 충돌하므로, **코드 자산만 재사용**했습니다.
> 나중에 PNG로 교체하려면 `game.js`의 `drawSprite()` 한 함수만 바꾸면 됩니다.

## GitHub Pages 배포
1. 이 저장소를 GitHub에 푸시합니다.
2. **Settings → Pages** 에서 Source 를 `Deploy from a branch` → `main` (root) 로 설정합니다.
3. 잠시 후 게임 주소로 접속:
   `https://<사용자명>.github.io/<저장소명>/street-cat-run/`

   (루트 `https://<사용자명>.github.io/<저장소명>/` 에서 바로 열고 싶다면
   `street-cat-run/`의 세 파일을 저장소 루트로 옮기면 됩니다.)

로컬 실행은 그냥 `street-cat-run/index.html`을 브라우저로 열면 됩니다.
